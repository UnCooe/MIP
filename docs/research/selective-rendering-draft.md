# Selective Rendering Draft

- Status: Draft
- Scope: default structure for generated context artifacts

## Goal

Make generated MIP context artifacts reflect the selective-consultation policy instead of flattening all memory into one equally weighted block.

## Current Draft Structure

- `Always-On Preferences`
- `Always-On Corrections And Work Style`
- `On-Demand Sections`
- deeper on-demand sections such as `Identity`, `Facts`, `Observations`, and `Pending Confirmation`

## Modes

- `selective` (default)
- `full`

## Why This Matters

Selective consultation cannot stay purely conceptual.
The default rendered artifact itself should guide agents toward the intended read behavior.
